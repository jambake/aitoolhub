# AI Prompts Cheat Sheet

## Copywriting Prompts

### Blog Post Intro
"Write a compelling introduction for a blog post about [TOPIC]. Hook the reader with a surprising statistic or question. Keep it under 100 words."

### Product Description
"Write a persuasive product description for [PRODUCT]. Focus on benefits, not features. Include a call to action. Target audience: [AUDIENCE]."

### Email Subject Lines
"Generate 10 email subject lines for [TYPE OF EMAIL]. Make them curiosity-inducing and click-worthy. Avoid clickbait."

## Image Generation Prompts

### Portrait Style
"Create a prompt for [SUBJECT] in [STYLE] style. Include: lighting details, color mood, composition, and mood. Describe in detail."

### Product Mockup
"Write a detailed prompt for a product photo of [PRODUCT] on [SURFACE]. Include: natural lighting, soft shadows, clean background."

## Coding Prompts

### Code Review
"Review this code and explain any bugs or issues: [PASTE CODE]. Suggest improvements for performance and readability."

### Explain Code
"Explain what this code does in simple terms: [PASTE CODE]. Break it down line by line."

## Productivity Prompts

### Meeting Notes
"Turn these meeting notes into actionable next steps: [PASTE NOTES]. Format as a bullet list with owners and deadlines."

### Summarize Long Text
"Summarize this [TEXT/ARTICLE/DOCUMENT] into 5 key points. Keep each point under 20 words."
