# Deploy Guide - AI Tool Hub

## Option 1: Cloudflare Pages (Recommended - Free)

### Step 1: Buy Domain
1. Go to **Cloudflare.com** → Register → Buy `aitoolhub.com`
2. Or use **Namecheap** or **GoDaddy** (~$15/year)

### Step 2: Connect to Cloudflare Pages
1. Go to **Cloudflare Dashboard** → **Pages** → **Create project**
2. Select **GitHub** (recommended) OR **Direct upload**
3. If Direct upload: drag & drop the `aitoolhub` folder

### Step 3: Configure
- **Project name:** aitoolhub
- **Production branch:** main
- **Build settings:** Leave empty (static HTML)
- **Domain:** Add your custom domain

**Done!** Your site will be live at `https://aitoolhub.com`

---

## Option 2: Netlify (Free Alternative)

1. Go to **Netlify.com** → Sign up
2. Drag & drop the `aitoolhub` folder onto the dashboard
3. Domain is auto-assigned (e.g., random-name.netlify.app)
4. Add custom domain in Settings → Domain management

---

## Option 3: GitHub Pages (Free)

1. Create repo `aitoolhub` on GitHub
2. Push the `aitoolhub` folder contents (not the folder itself)
3. Go to **Settings** → **Pages** → Source: main branch
4. Your site: `https://yourusername.github.io/aitoolhub/`

---
